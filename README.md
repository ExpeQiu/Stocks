# 股票分析仪表盘

一个功能丰富的股票分析仪表盘，使用 React + TypeScript + Vite 构建，提供实时股票数据可视化和分析功能。

## 功能特点

- 📈 实时股票价格图表展示
- 💹 多维度收益分析
- 📊 成交量分析
- 🌍 全球市场指标
- 📉 技术指标展示
- 🔄 市场情绪分析
- 💰 资金流向分析

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Recharts (图表库)
- Lucide React (图标库)
- Date-fns (日期处理)
